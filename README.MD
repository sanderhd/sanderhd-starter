## Usage:
You can use my CLI by running the following command:

```bash
npm install -g sanderhd-starter
```

After this, you can use 

```bash
sanderhd-starter
```
anywhere you want!

* DiscordJS Simple - Using the basic `!` commands.
* DiscordJS Advanced - Using slash commands.
